## for vs while

for: 반복가능한 애들을 하나 하나 꺼내줌. 다 꺼내면 종료

while: 조건이 참일때 실행됨. 종료조건을 설정해야 함



```python
a = {'key': 'value'}
for k, v in a.items():
    print(k, v)
```
